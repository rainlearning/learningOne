## headers属性介绍

-Remote Address :远程服务器

-request url：请求的url

-request Method:get 请求方式 大概8种

-status code :200 ok 响应状态码

-connection:keep-alive 保持连接

-date 时间  

-transfer-Encoding:chunked 

-accept 客户端接收的类型

-accept-encoding ：压缩

-accept-language  语言

-cache-control :缓存的规范
